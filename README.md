# 🔎 Nmap Automation Tool

## 📌 Overview
This tool automates Nmap scanning for reconnaissance and saves the results into a text file.

## 🛠️ Features
- Runs Nmap with service detection and OS detection
- Saves results automatically to scan_results.txt
- Simple one‑command usage

## 🚀 How to Run
python3 nmap_auto.py <target>

- Python scripting
